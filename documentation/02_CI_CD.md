### ***What is the GitHub actions***
A workflow Automation service by github 
	Automate all kind of repository related processes and actions 

### ***What is CI/CD***
1. CI -  Continuous integration 
	- code changes are automatically build tested and merged with existing code 
2. CD - Continuous delivery
	- After integration new app or package version are published automatically ( publishing the new code after the code has been built tested  )
